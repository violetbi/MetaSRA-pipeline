Links to useful resources

Sequence Read Archive

https://www.ncbi.nlm.nih.gov/sra/

Ontologies

MetaSRA annotates samples with the following ontologies:

  • Cell Ontology
  • Cellosaurus
  • Disease Ontology
  • Experimental Factor Ontology
  • Uberon
  • Unit Ontology
These ontologies can be searched using the Ontology Lookup Service.

SRAdb

Raw metadata in the SRA has been collected by the SRAdb project:
https://bioconductor.org/packages/release/bioc/html/SRAdb.html

Publication

A pre-print describing the MetaSRA can be found at the bioRxiv:
http://biorxiv.org/content/early/2016/12/19/090506

Source code

The code used to construct the MetaSRA is maintained on GitHub:
